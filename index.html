Minimalist and intuitive MQTT broker and client written entirely in C#

Minimalist and intuitive asynchronous MQTT messaging broker and client written entirely in C#

Installation

Server (broker) package:

Install-Package Hermes.Server

Client package:

Install-Package Hermes.Client

To debug with full symbols and sources the above packages, setup SymbolSource in the Visual Studio debugging options as explained in the SymbolSource site
